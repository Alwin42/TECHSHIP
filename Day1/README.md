
# Day 1: Getting Started with JavaScript

### 📝 Overview
Welcome to Day 1 of my TECHSHIP internship! This is where my journey begins with learning JavaScript fundamentals. Today's focus is on understanding the basics of console output in JavaScript.

### 🎯 Learning Objectives
- Understanding `console.log()` function
- Printing output to the console
- Writing basic JavaScript statements

### 💻 Code Explanation

```javascript
console.log("Alwin Emmanuel");
console.log("AISAT Engineering College");
console.log("Computer Science and Engineering");
console.log("Welcome to TECHSHIP ");
```

#### What does this code do?

1. **`console.log()`** - This is a built-in JavaScript function used to print messages to the browser's console or terminal.

2. **Line-by-line breakdown:**
   - **Line 1:** Prints my name: "Alwin Emmanuel"
   - **Line 2:** Prints my college name: "AISAT Engineering College"
   - **Line 3:** Prints my department: "Computer Science and Engineering"
   - **Line 4:** Prints a welcome message: "Welcome to TECHSHIP"

### 🏃‍♂️ How to Run This Code

#### Method 1: Browser Console
1. Open your browser (Chrome, Firefox, etc.)
2. Press `F12` or right-click and select "Inspect"
3. Go to the "Console" tab
4. Copy and paste the code
5. Press `Enter`

#### Method 2: Node.js
1. Make sure you have Node.js installed
2. Create a file named `day1.js`
3. Copy the code into the file
4. Open terminal in the same directory
5. Run: `node day1.js`

### 📤 Expected Output
```
Alwin Emmanuel
AISAT Engineering College
Computer Science and Engineering
Welcome to TECHSHIP
```

### ✨ Key Takeaways
- `console.log()` is used for debugging and displaying output
- Each `console.log()` statement prints on a new line
- Strings in JavaScript must be enclosed in quotes (single `' '` or double `" "`)
